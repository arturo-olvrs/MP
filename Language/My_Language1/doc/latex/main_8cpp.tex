\doxysubsection{Detailed Description}
This program reads a text (without spaces) with a undefined number of characters and a text with two characters (bigram). It finds the bigrams contained in the first text, storing them in an array of \mbox{\hyperlink{classBigram}{Bigram}}. After that, the bigrams of the array are shown in the standard output. Then it converts to uppercase the bigrams in the array that are equals to the bigram of the second text. Finally the bigrams of the array are shown again in the standard output.
